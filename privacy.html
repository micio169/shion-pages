개인정보 보호 정책

시온「シオン」이 어떤 데이터를 사용하고 저장하는지 설명합니다.

1. 수집 및 저장하는 정보

시온「シオン」은 서버별 설정을 위해 서버 ID, 로그 채널 ID, 티켓 카테고리 ID, 관리자 역할 ID를 저장할 수 있습니다.

2. 처리할 수 있는 정보

로그 기능을 위해 메시지 삭제, 메시지 수정, 멤버 입장, 멤버 퇴장 같은 서버 이벤트 정보를 처리할 수 있습니다. 메시지 내용은 서버 관리자가 설정한 로그 채널에 기록될 수 있습니다.

3. 정보 사용 목적

저장된 정보는 로그 전송, 티켓 채널 생성, 관리자 권한 적용, 서버별 설정 유지에만 사용됩니다.

4. 정보 공유

시온「シオン」은 저장된 설정 정보를 판매하거나 광고 목적으로 공유하지 않습니다. Discord API 제공과 봇 운영에 필요한 범위 안에서만 정보를 처리합니다.

5. 보관 기간

서버 설정 정보는 봇이 서버에서 사용되는 동안 보관될 수 있습니다. 서버 관리자가 삭제를 요청하거나 봇 운영자가 서비스를 종료하는 경우 삭제될 수 있습니다.

6. 삭제 요청

서버 관리자는 저장된 서버 설정 삭제를 요청할 수 있습니다. 삭제 요청 시 서버 ID 확인이 필요할 수 있습니다.

7. 보안

봇 토큰과 설정 파일은 운영 환경에서 관리됩니다. 단, 서버 운영자는 로그 채널 권한을 적절히 제한하여 민감한 기록이 불필요하게 노출되지 않도록 해야 합니다.

8. 정책 변경

기능 변경, Discord 정책 변경, 운영 방식 변경에 따라 이 정책은 수정될 수 있습니다.
