Core features

서버 운영에 필요한 것만 선명하게

LOG

디스코드 로그

메시지 삭제, 메시지 수정, 멤버 입장과 퇴장 기록을 지정한 로그 채널에 남깁니다.

TIX

티켓 문의

버튼 하나로 개인 문의 채널을 만들고, 관리자 역할에게만 확인 권한을 부여합니다.

MOD

관리 명령어

메시지 정리 같은 기본 관리 기능을 슬래시 명령어로 빠르게 실행할 수 있습니다.

Quick setup

초대 후 서버에서 바로 설정

시온은 한 서버에만 묶이지 않습니다. 각 서버 관리자가 원하는 로그 채널, 티켓 카테고리, 관리자 역할을 직접 지정하면 그 서버 설정으로 작동합니다.

/setup log_channel, ticket_category, staff_role
/ticket-panel 티켓 버튼 전송
/clear 메시지 정리

For admins

서버별 설정, 필요한 데이터만

서버 ID, 로그 채널 ID, 티켓 카테고리 ID, 관리자 역할 ID만 설정 저장에 사용합니다.

메시지 로그 기능은 서버 관리자가 지정한 로그 채널 안에서만 기록됩니다.

서버에서 봇을 제거하거나 운영자에게 요청하면 저장된 설정 삭제를 요청할 수 있습니다.
